# Инструкция по использованию мини-приложения «Вишлист с Wildberries»
## Как открыть приложение
Перейдите по ссылке https://vk.com/app54477328. Приложение откроется в окне ВКонтакте, авторизация произойдет автоматически (потребуется разрешение на доступ к данным профиля).

## Экран «Мой вишлист» (главный)
В верхней части отображается ваш аватар, имя пользователя и количество желаний.

Под ними расположена лента желаний в виде квадратных карточек с изображениями товаров.

Если список пуст, вы увидите сообщение «Список желаний пуст» и кнопку для добавления первого желания.

## Добавление желания
Нажмите на круглую кнопку с плюсом ➕ в правой части шапки.

В открывшемся окне вставьте ссылку на товар с Wildberries (например, https://www.wildberries.ru/catalog/12345678/detail.aspx). Можно вставлять ссылку вместе с текстом (например, название товара, скопированное из приложения WB).

Нажмите «Добавить». Приложение автоматически найдет изображение товара и добавит карточку в начало ленты.

Если товар не найден, появится экран «Такой товар не найден» с предложением проверить ссылку.

## Удаление желания
На карточке товара в правом верхнем углу нажмите на иконку корзины 🗑️.

Желание будет удалено без дополнительного подтверждения.

## Переход к товару
Нажмите на любую карточку желания – откроется страница товара на Wildberries во внешнем браузере.

## Поделиться списком желаний
Нажмите на иконку «Поделиться» 🔗 в шапке (рядом с кнопкой добавления).

Будет сформировано сообщение со ссылками на все ваши желания, которое можно отправить другу или в любой чат ВКонтакте.

## Экран «Друзья»
Переключитесь на этот экран с помощью нижней панели навигации.

В данный момент раздел находится в разработке – здесь отображается заглушка с информацией о том, что скоро появятся вишлисты друзей.

В будущем здесь можно будет просматривать списки желаний других пользователей.

## Сохранение данных
Все желания сохраняются в облаке VK Storage и автоматически подгружаются при следующем входе с любого устройства.

Список не очищается автоматически – вы сами управляете им, добавляя и удаляя желания.

При добавлении товара изображение кешируется локально для ускорения загрузки при повторных посещениях.
